Eventuate

Eventuate is a toolkit for building distributed, highly-available and partition-tolerant event-sourced applications. It is written in Scala and built on top of Akka, a toolkit for building highly concurrent, distributed, and resilient message-driven applications on the JVM. Eventuate also provides an implementation of operation-based CRDTs. Learn more in the project documentation and in the API docs.
