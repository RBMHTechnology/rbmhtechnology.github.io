## Lists
### Unordered Lists
Starts with *

* List one
* List two

#### Nested Unordered List
* List one
    * List one one
    * List one two
* List two
* List three
    1. List one one
    + List one two


###Ordered List
Starts with 1 and follows +

1. List-entry number 1
+ List-entry number 2
+ List-entry number 3

#### Nested Ordered List
1. List-entry number 1
+ List-entry number 2
    1. List-entry number 2.1
    + List-entry number 2.2
+ List-entry number 3
    * List-entry number 2.1
    * List-entry number 2.2
